# Deep Research Workflow Detail

1. `web_search` broad query first (1-2 words), then narrow
2. Pull 3-5 promising URLs from results
3. `fetch_docs`/`fetch_url` each one that looks authoritative (official docs, primary sources
   over aggregators/forums)
4. Cross-check: if two sources disagree, search again with more specific terms rather than
   picking one arbitrarily
5. Synthesize in your own words — don't just concatenate snippets
6. Cite sources by name/URL in the final answer
